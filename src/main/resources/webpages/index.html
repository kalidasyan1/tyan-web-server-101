Welcome! This is a simple file server built with Netty demonstrating static file serving capabilities.

Server Information

  • Framework: Netty (High-performance NIO framework)
  • Port: 8085
  • Root Directory: Project root directory
  • Supported Files: HTML, CSS, JS, Images, and more

Features Demonstrated

Chunked File Transfer: Efficiently serves large files using Netty's ChunkedFile handler
MIME Type Detection: Automatically sets appropriate Content-Type headers based on file extensions
Error Handling: Proper HTTP error responses for missing files or invalid requests
Keep-Alive Support: HTTP/1.1 persistent connections for better performance

Architecture

This file server uses Netty's EventLoop architecture for high-performance I/O operations:

  • Single EventLoop Thread: All I/O operations for each connection happen on the same thread
  • Non-blocking I/O: Handles multiple concurrent connections efficiently
  • Zero-copy Transfer: Uses ChunkedFile for efficient file streaming
Note: This server serves files from the project root directory. Make sure you understand the security implications in production environments.
